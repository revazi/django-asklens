# Demo query ideas

These examples are intended for deterministic dummy-provider demos and later provider evaluation fixtures.

- Show orders by status as a bar chart.
- How many orders were placed this month?
- Show revenue by month as a line chart.
- List pending orders from the last 30 days.
- Show average order value by month.

Demo queries should use registered resources and metrics only. They should not require sample database rows to be sent to an LLM provider.
