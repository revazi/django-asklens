# Django AskLens

Django AskLens is a planned reusable Django + DRF package for safe natural-language querying over explicitly registered Django models.

Status: early planning. The package scaffold has not been added yet.

Planned names:

- GitHub repository: `django-asklens`
- Python package distribution: `django-asklens`
- Python import package: `django_asklens`
- Django app label: `asklens`
